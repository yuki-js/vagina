# プライバシーポリシー & データ取り扱いガイド

## 概要

VAGINA (Voice AGI Notepad Agent) は、ユーザーのプライバシーを尊重し、最小限のデータ収集でサービスを提供します。

## 収集するデータ

### 音声データ
- **収集**: 通話中のみマイクから音声を取得
- **送信**: リアルタイムで OpenAI API に送信
- **保存**: 端末内には保存しません
- **用途**: AI アシスタントとの会話のみ

### API キー
- **収集**: ユーザーが設定画面で入力
- **保存**: 端末内のセキュアストレージ（Android: EncryptedSharedPreferences, iOS: Keychain）
- **送信**: OpenAI API への認証にのみ使用
- **第三者共有**: なし

### アプリ設定
- **収集**: マイクゲイン、音声選択などの設定
- **保存**: 端末内のローカルストレージ
- **送信**: 送信しません
- **用途**: ユーザー体験のカスタマイズ

## 収集しないデータ

- 通話履歴・会話内容の保存
- 位置情報
- 連絡先
- 端末識別子
- 使用状況分析データ

## OpenAI のデータ利用

音声データは OpenAI のサーバーに送信されます。OpenAI のデータ利用ポリシーについては、[OpenAI Privacy Policy](https://openai.com/policies/privacy-policy) を参照してください。

### 重要事項
- OpenAI API 経由のデータは、デフォルトで OpenAI のモデル学習には使用されません (API Terms of Service)
- ただし、サービス改善のため一定期間ログが保持される可能性があります

## セキュリティ対策

### 通信
- WebSocket 接続は TLS (wss://) で暗号化
- API キーはクエリパラメータとして送信（OpenAI 仕様）

### ストレージ
- API キーは OS レベルのセキュアストレージに保存
- 暗号化キーはハードウェアセキュリティモジュールで管理（対応端末）

### アプリケーション
- API キーはソースコードにハードコードしない
- `.env` ファイルは `.gitignore` で除外

## ユーザーの権利

### データ削除
設定画面から API キーを削除できます。音声データは端末に保存されていないため、削除対象はありません。

### データエクスポート
ローカル保存データのエクスポート機能は現在提供していません。
