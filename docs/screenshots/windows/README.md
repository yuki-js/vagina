# Windows スクリーンショット

このディレクトリには、Windows 版アプリケーションのスクリーンショットを配置します。

## 必要なスクリーンショット

1. `01_startup.png` - 起動画面
2. `02_main_idle.png` - メイン画面（待機中）
3. `03_recording.png` - 音声入力中
4. `04_conversation.png` - 会話履歴
5. `05_settings.png` - 設定画面

## 注意事項

- API キーやその他の機密情報は伏せ字にしてください
- PNG 形式で保存してください
- 解像度は 1920x1080 以上を推奨

詳細なテスト手順については [WINDOWS_TESTING.md](../../WINDOWS_TESTING.md) を参照してください。
